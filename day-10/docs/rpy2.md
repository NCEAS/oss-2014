#Rpy2 - Tutorial

![rpy2](images/rpy2.png)

Several tutuorials for the Rpy2 Python package can be found at [rpy.sourceforge.net](http://rpy.sourceforge.net/rpy2/doc-dev/html/introduction.html)

Including [graphics](http://rpy.sourceforge.net/rpy2/doc-dev/html/graphics.html#graphics) tutorials

* for using lattice, box plot, ggplot2, etc.

If you plan to follow the tutorials for Rpy2 using the xubuntu VM that we installed in class you should install the Rpy2 package from Terminal Emulator

###Install Method using apt-get


```
oss@oss-VirtualBox:~$ apt-cache search rpy2
python-rpy2 - Python interface to the GNU R language and environment (version 2)
```
